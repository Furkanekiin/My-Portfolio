## Subscription lifecycle

Webhook olaylarını yakalayarak kullanıcıların abonelik durumlarını veritabanında güncel tutmak ve yetkilendirmeleri buna göre yönetmek gerekir.

<CodeBlock
    codeHeight={24}
    marginBottom="16"
    codes={[
  {
    code:
`const stripeConfig = {
    checkout: {
      paymentMethodTypes: ['card'],
      lineItems: [{ price: 'price_id', quantity: 1 }],
      mode: 'subscription',
      allowPromotionCodes: true
    },
    webhook: {
      events: [
        'customer.subscription.created',
        'customer.subscription.updated',
        'customer.subscription.deleted',
        'invoice.payment_succeeded'
      ]
    },
    portal: {
      enabled: true,
      returnUrl: '/dashboard/settings'
    }
};`, 
    language: "tsx",
    label: "src/config/stripe.js"
  }
]} />
