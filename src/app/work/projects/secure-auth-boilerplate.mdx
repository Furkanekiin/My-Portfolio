# Secure Authentication Boilerplate

Projelerinizde hızlıca entegre edebileceğiniz, yüksek güvenlik standartlarına sahip kimlik doğrulama şablonu.

## Özellikler
- JWT tabanlı güvenli oturum yönetimi
- Şifreleme ve güvenlik katmanları
- PostgreSQL entegre kullanıcı veritabanı şeması
